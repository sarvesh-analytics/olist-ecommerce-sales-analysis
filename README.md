# Olist E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes the Brazilian Olist E-Commerce dataset using PostgreSQL and Power BI. The goal of the project is to transform raw e-commerce data into meaningful business insights through data modeling, SQL analysis, and interactive dashboards.

## Tools Used

* PostgreSQL
* SQL
* Power BI
* DAX

## Database Design

The database was designed using 9 interconnected tables:

* Customers
* Orders
* Order Items
* Products
* Sellers
* Payments
* Reviews
* Geolocation
* Product Category Translation

Primary keys and foreign key relationships were implemented to maintain data integrity.

## Data Processing

* Imported CSV datasets into PostgreSQL.
* Created relational database schema.
* Performed data cleaning and validation.
* Created SQL views for reporting and analytics.
* Connected PostgreSQL to Power BI using Direct Query.
